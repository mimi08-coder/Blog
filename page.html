1. Introduction

Binary search is a powerful algorithm used to quickly find elements in a sorted array, or to search over the space of possible answers in optimization problems.

2. When to Use

  • The array or search space is sorted or monotonic
  • You need to find a minimum/maximum that satisfies a condition
  • You want to optimize time complexity from O(n) to O(log n)

3. Binary Search Template


// C++ Binary Search Template
int l = 0, r = n - 1;
while (l <= r) {
    int mid = (l + r) / 2;
    if (a[mid] == target) return mid;
    else if (a[mid] < target) l = mid + 1;
    else r = mid - 1;
}
return -1;
